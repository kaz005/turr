# プロジェクト仕様書

## プロジェクト名
Turr

## バージョン
0.1.0

## 概要
Turrは、Next.jsを使用したウェブアプリケーションで、ツアーの詳細を表示し、予約を行うことができます。ReactとTailwind CSSを使用して、モダンでレスポンシブなUIを提供します。

## 使用技術
- Next.js
- React
- Tailwind CSS
- TypeScript
